# 📊 Online Retail Sales Analysis Dashboard

An end-to-end data analytics project that explores customer purchasing behavior using the Online Retail dataset. The project demonstrates data cleaning, feature engineering, exploratory data analysis (EDA), interactive visualizations, and dashboard development using Python.

---

## 📌 Project Overview

This project analyzes transactional sales data from an online retail store to uncover business insights such as:

- Revenue trends over time
- Sales performance by country
- Customer purchasing behavior
- Best-selling products
- Order distribution
- Interactive business dashboard

The project was developed using **Python, Pandas, Plotly, and Dash**.

## 📊 Key Insights

- United Kingdom generated the highest revenue.
- Revenue fluctuates throughout the year with seasonal peaks.
- A small number of customers contribute a large proportion of total sales.
- Several products consistently outperform others in revenue generation.

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/segbo/Online-Retail-Sales-Analysis.git
```

Navigate into the project

```bash
cd Online-Retail-Sales-Analysis
```

Install dependencies

```bash
pip install pandas numpy plotly dash
```

Run the notebook

```bash
jupyter notebook
```

or launch the Dash application

```bash
python app.py
```
